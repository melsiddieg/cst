About this demo

This model was trained on 100K Arabic sentences using Context-Sensitive Tokenization (CST) — a morphology-aware tokenizer for Arabic. It runs fully in your browser via ONNX Runtime WebAssembly (no server). Type an Arabic phrase and the model predicts what comes next.
